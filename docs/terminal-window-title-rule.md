# Terminal Window Title Rule

Правило для терминальной сцены NetDen:

1. Каждое новое inline-окно должно иметь свой заголовок в верхней зоне сцены.
2. Заголовки централизованно задаются в маппинге `window -> title` внутри терминального компонента.
3. При выходе из любого окна и возврате в базовую консоль заголовок обязательно возвращается к `NetDen`.
4. Перед добавлением нового окна разработчик обязан добавить:
   - идентификатор окна,
   - title в общий маппинг,
   - переключение title при открытии/закрытии.
